#### Соображения по лабораторной работе №4

Для четвёртой лабораторной планирую разработать **eDSL для конечных автоматов**, который позволит описывать состояния, переходы и генерировать `.dot`-файлы для визуализации в Graphviz. Это задание даст возможность продемонстрировать выразительность Haskell в построении предметно-ориентированных языков и использовании монадо-подобных структур.

#### Инструментальные средства

* **Компилятор:** GHC (Glasgow Haskell Compiler)
* **Система сборки:** Stack
* **Тестирование:** Hspec, QuickCheck
* **Форматирование и линтинг:** Ormolu, HLint
* **CI/CD:** GitHub Actions
* **Система контроля версий:** Git (GitHub)

#### Использованные источники

1. Miran Lipovača — *Learn You a Haskell for Great Good!*
2. Graham Hutton — *Programming in Haskell*
3. Haskell Wiki
4. Документация по GHC и Stack
5. HLint и Ormolu GitHub-репозитории

#### Выбор книги

Для изучения языка и подготовки к зачёту выбрана книга **Miran Lipovača — “Learn You a Haskell for Great Good!”**, так как она даёт систематичное и интуитивное введение в функциональное программирование на Haskell и охватывает все темы, необходимые для лабораторного курса.
